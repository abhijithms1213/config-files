## Question :
- ######  *How to host website on an ip using Vagrant and centos?*  
## Explanatio**n**
---
- use */var/www/html/* dir for host the website
	- put index.html and etc files on this folder 
	- use **tooplate.com** for template files
	- check ip using [[ip address show]] and check httpd enabled or not? then go to a webbrowser and check with *http//:ip*
