## Question :
- ######  *how to write whoam command's output to a file*  
## Explanatio**n**
---
- *uptime > filename.txt*
- *whoami > logTime.txt*
- *ls > logTime.txt*
	- it all log's and write to a file 
## Disadvantages
- it will rewrite the current content of the file ,that can be avoided by using [[>> sign]].
