# Zettelkasten Vault Setup Guide

## 1. Install These Plugins in Obsidian
- Zettelkasten Prefixer
- QuickAdd
- Dataview
- Periodic Notes
- Spaced Repetition
- Tag Wrangler

## 2. Folder Structure
- **00 Inbox** → Quick capture for raw ideas
- **01 Literature Notes** → Notes from books, videos, courses
- **02 Permanent Notes** → Atomic Zettelkasten notes
- **03 Project Notes** → Active project-related notes
- **04 Resources** → References and materials
- **attachments** → Images, PDFs, etc.

## 3. Workflow
1. Capture ideas in **Inbox**
2. Process into **Permanent Notes** (one idea per note)
3. Link notes using [[Note Name]]
4. Review with Spaced Repetition

## 4. Daily Notes
Use **Periodic Notes** plugin to generate daily notes from `templates/daily_note.md`.
