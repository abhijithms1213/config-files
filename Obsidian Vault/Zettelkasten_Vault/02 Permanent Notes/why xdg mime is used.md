## Question :
- ######  *What's the purpose of using it*  
## Explanatio**n**
---
- xdg mime is get from [[xdg-utils why xdg utils are important in their DE's]] , so it's a tool which provided by xdg utils
- xdg mime provides to open anything in particular tool based on MIME type
- eg:
	- open a text file in notepad
	- 
- or we can set *default* to open anything's open in particular tool permanently
```bash
xdg-mime default kitty.desktop x-scheme-handler/terminal
```
here **MIME** type is x-scheme-handler/terminal and open in kitty.desktop
