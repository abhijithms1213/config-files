## Question :
- ######  *Why syncfolder is using and how to write it*  
## Explanatio**n**
---
- Why?
	- because if we want to attach a system's file or folder to a particular folder in vBox.
- How to use it
```js
config.vm.synced_folder "./firstPage/scripts", "/opt/scripts"
```
here first *" "* is system's folder and where to attach is second one, here attach to /opt/scripts
