## Question :
- ######  *How to set a file's permission using numbers?*  
## Explanatio**n**
---
```bash
chmod 644 [file name]

read     = 4
write    = 2
execute  = 1
owner    = read | write = 6
group    = read         = 4
other    = read         = 4
```
