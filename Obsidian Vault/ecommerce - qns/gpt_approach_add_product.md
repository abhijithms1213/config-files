# How to Add Products in Shops Without Complication

## Core Idea

Instead of manually adding every single product for every shop, we use a **Category-Based Product Mapping System**.

This works for:

- Petti kada
- Grocery shop
- Tea shop
- Bakery
- Vegetable shop
- Fish/Meat shop
- Juice shop
- Flower shop

These are usually **hybrid stores** because they sell products from multiple categories.

Example:

- Shampoo hanging packets
- Washing powder
- Mixture
- Chips
- Mango
- Naranga vellam
- Tea
- Small snacks

So instead of adding products one by one, we attach **product categories/modules** to the shop.

- Petti kada

# Problem Case

# Example: Payar Category Problem

Suppose:

Shop has only:

- Valli Payar

But category attached is:

## `Payar Gang`

Which includes:

- Valli Payar
- Beans
- Payar
- Others

Problem:

Seller may receive orders for products they don’t have from this category.

---

# Solution — Case 1

## Seller Handles “Not Available”

When order arrives for unavailable product:

Seller taps:

## Option 1

`Cancel → We don't have this product`

### Result

System permanently ignores this shop for that product next time.

---

## Option 2

`Cancel → Currently unavailable`

### Result

System ignores only for this order.

Later it can still receive that product order.

---

## Re-Adding Ignored Products Later

Seller app should have:

# Ignored Products Page

Features:

- Categorized ignored list
- Seller can reopen products later
- Remove ignored status manually

This solves restocking problems.

---

# Expanding Shop Later

If seller adds new products:

Example:

Earlier only sold vegetables

Now also sells:

- Shampoo
- Snacks
- Oils

Then seller should have:

# Add New Category Option

Methods:

## Option 1

Contact Us

Admin adds category

---

## Option 2

Seller Self-Service

Seller opens:

`Manage Categories`

and selects:

- Shampoo
- Snacks
- Oil

etc.

This is much better.

---

# Better Improvement

When seller selects a category:

They should also specify:

## Which exact products exist

Example:

Category = Shampoo

Seller selects:

- Clinic Plus
- Sunsilk
- Head & Shoulders

Only these products receive orders.

This avoids unnecessary cancellations.

---

# Solution — Case 2 (Best Approach)

## Do NOT Add Full Category Always

If seller has only 1–2 items from a category:

Do not attach the full category.

Instead:

## Only Add Available Products

Example:

Category = Payar Gang

Seller only has:

- Valli Payar

Then only enable:

- Valli Payar

Do not enable:

- Beans
- Green Payar
- Other items

This is the best production approach.
